# Clientes

Como achar e fechar clientes.

---

## Antes de Prospectar

> "Mensagem boa fala de DOR, não de tecnologia."

**Checklist:**
- [ ] Escolhi UM setor (ver `01-setores/guia-setores.md`)
- [ ] Sei qual DOR vou atacar
- [ ] Tenho lista de 50 leads
- [ ] Sei as palavras que o cliente usa (não as minhas)

**Palavras PROIBIDAS na primeira mensagem:**
- IA, Inteligência Artificial, Automação
- Bot, Agente, RAG, LLM
- Sistema, Plataforma, Ferramenta

**Palavras que FUNCIONAM:**
- Problema, dor, frustração
- Tempo, custo, cliente, lead
- Resultado, meta, objetivo

## Onde Encontrar Leads

### Por Setor

| Setor | Onde encontrar | Ferramentas |
|-------|----------------|-------------|
| **Advocacia** | OAB, LinkedIn, Econodata | Sales Navigator, Google Maps |
| **Clínicas** | Google Maps, Doctoralia | Instagram local, LinkedIn |
| **Imobiliárias** | CRECI, ZAP Imóveis, Viva Real | LinkedIn, Google Maps |
| **E-commerce** | Instagram shops, Shopify/Nuvemshop | BuiltWith, SimilarWeb |
| **Agências** | LinkedIn, Clutch, Google | Sales Navigator |
| **Contabilidade** | CRC, Google Maps, LinkedIn | Indicações |

### Processo para Montar Lista

1. **Google Maps** → Busca "[setor] + [cidade]"
2. **LinkedIn** → Filtra cargo + setor + tamanho empresa
3. **Instagram** → Hashtags do setor, perfis comerciais
4. **Indicação** → Pede 3 nomes pra quem você conhece

**Meta:** 50 leads ANTES de mandar primeira mensagem.

## Canais de Aquisição

### 1. Cold Outreach (WhatsApp/LinkedIn)
**O que é:** Mandar mensagem direta pra empresas do nicho.

**Prós:** Controle total, resultado rápido se funcionar
**Contras:** Taxa de resposta baixa, precisa de volume

**Meta:** 25 mensagens/semana

### 2. Rede Pessoal
**O que é:** Pedir indicações pra quem você conhece.

**Prós:** Confiança já existe, conversão alta
**Contras:** Escala limitada

**Ação:** Liste 20 pessoas que podem indicar. Mande mensagem.

### 3. Conteúdo (LinkedIn)
**O que é:** Postar sobre problemas do setor, atrair inbound.

**Prós:** Escala, autoridade
**Contras:** Demora pra dar resultado

**Meta:** 2 posts/semana (SÓ SE bateu meta de outreach)

### 4. Comunidades
**O que é:** Participar de grupos onde empresários do nicho estão.

**Prós:** Acesso direto ao público
**Contras:** Precisa agregar valor antes de vender

## Scripts de Prospecção

### Abertura Direta (Hormozi)

```
Ainda [dor em forma de pergunta]?
```

**Exemplos por dor:**

| Dor | Mensagem |
|-----|----------|
| Atendimento lento | "Ainda perde cliente por demorar pra responder no WhatsApp?" |
| Equipe sobrecarregada | "Seu time ainda gasta horas em tarefas que podiam ser automáticas?" |
| Informação espalhada | "Ainda perde tempo procurando documento que deveria estar na mão?" |
| Prospecção manual | "Seus vendedores ainda gastam mais tempo pesquisando do que vendendo?" |

### Abertura ACA (leads mais frios)

```
Oi [nome], vi que você [algo específico do perfil].

[Observação genuína sobre a empresa].

Vocês ainda [dor em forma de pergunta]?
```

**Exemplo:**
```
Oi Marcos, vi que você tem uma imobiliária em Curitiba.

Parabéns pelo crescimento - vi que abriram mais uma unidade.

Vocês ainda perdem leads por demorar pra responder no WhatsApp?
```

### Follow-up 1 (3 dias depois)

```
[Nome], só checando se viu minha mensagem.

Se [dor resumida] não é um problema pra vocês, sem problema - só me avisa.
```

### Follow-up 2 (7 dias depois)

```
Última mensagem, [nome].

Se [dor] não é prioridade agora, entendo totalmente.

Fico à disposição se mudar.
```

**Regra:** 2 follow-ups. Depois disso, desiste.

## Quando Responderem

### Se disser "sim, tenho esse problema"

```
Ótimo. Posso te mandar um vídeo de 2min mostrando como outras [empresas do setor] resolveram isso?
```

### Se perguntar "o que você faz?"

```
Ajudo [setor] a [resultado] sem [esforço/tempo].

Exemplo: uma [empresa similar] conseguiu [resultado específico] em [tempo].

Quer que eu mostre como?
```

### Se disser "não tenho esse problema"

```
Ótimo, fico feliz que esteja resolvido.

Se mudar no futuro, fico à disposição. Abraço!
```

## Regras

1. **50 leads na lista ANTES de mandar primeira mensagem**
2. **Prospectar ANTES de criar conteúdo ou estudar**
3. **50 prospecções no setor antes de trocar**
4. **2 follow-ups, depois desiste**
5. **Atualizar pipeline toda semana**

---

## Rotina Mínima

| Dia | Ação |
|-----|------|
| Segunda | 10 mensagens novas |
| Terça | Follow-ups + 5 novas |
| Quarta | 10 mensagens novas |
| Quinta | Follow-ups + 5 novas |
| Sexta | Follow-ups + revisar pipeline |

**Total:** 30 novas + follow-ups = meta batida

## Teste da Mensagem

Antes de mandar, pergunte:
> "Essa mensagem faria sentido pra alguém que não sabe o que é IA?"

Se não, reescreve.

## Ainda não sabe como quer fazer?

Tá ok. Comece pelo mais simples:

1. Escolhe UM setor (ver `01-setores/guia-setores.md`)
2. Escolhe UMA dor desse setor
3. Acha 50 empresas
4. Manda 5 mensagens por dia
5. Vê o que acontece
6. Ajusta com feedback real

Não precisa de estratégia perfeita. Precisa de dados.
