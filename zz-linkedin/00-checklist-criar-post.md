# Checklist: Criar Post LinkedIn

> Processo interativo para criar posts. Use com Claude.

**Referências:**
- [tipos-conteudo.md](tipos-conteudo.md) - pilares, frameworks, hooks, banco de ideias
- [guia-conteudo-linkedin.md](guia-conteudo-linkedin.md) - prompts, ferramentas, engajamento

---

## Pré-requisito

- [ ] Mandei 10 mensagens de prospecção hoje

---

## 1. Validar Ideia vs ICP

Antes de continuar, responda:
- Qual a ideia do post?
- Esse post atrai meu ICP? (empresários de empresas médias)
- Qual dor/desejo ele endereça?

**ICP:** Empresários de empresas médias no Brasil
**Dores:** Medo de ficar pra trás com IA / Acham que IA é só pra tech

Se não atrai ICP → descarta e pega outra ideia

---

## 2. Escolher Pilar

| Pilar | Objetivo | Formato Ideal |
|-------|----------|---------------|
| Build in Public | Mostrar que sei fazer | Vídeo/Screenshot |
| Authoritative | Posicionar como referência | Carrossel/Infográfico |
| Personal | Gerar identificação | Foto + texto |

---

## 3. Escolher Framework

| Framework | Quando Usar | Estrutura |
|-----------|-------------|-----------|
| PAS | Resolver problema | Problema → Agitar → Solução |
| BBA | Mostrar transformação | Antes → Ponte → Depois |
| CPF | Case studies | Afirmação → Prova → Filtro |
| AIDA | Posts pessoais | Atenção → Interesse → Desejo → Ação |

---

## 4. Brain Dump

Fale tudo que sabe sobre o tema (10min, pode usar Whisper)

---

## 5. Criar Hook (2 linhas)

**Tipos de hook:**

| Tipo | Exemplo |
|------|---------|
| Fato | "80% dos empresários têm medo de ficar pra trás com IA. Só 12% estão fazendo algo." |
| Credibilidade | "Automatizei o atendimento de X clientes sem contratar ninguém. Levou 2 semanas." |
| Afirmação | "IA não é só pra empresas de tech. É pra quem não quer trabalhar até meia-noite." |

**Checklist do hook:**
- [ ] Para o scroll?
- [ ] Direcionado ao ICP?
- [ ] Gera curiosidade?

---

## 6. Estruturar Body

Usando o framework escolhido, estruture o corpo.

**Checklist:**
- [ ] Linguagem assertiva (sem "eu acho", "talvez")
- [ ] Voz ativa
- [ ] Números no lugar de adjetivos vagos
- [ ] Frases curtas e escaneáveis

---

## 7. Definir CTA

- [ ] Tem lead magnet ou pergunta?
- [ ] Deixa claro quem você ajuda?

---

## 8. Visual

| Formato | Quando usar |
|---------|-------------|
| Vídeo curto | Build in Public (demos) |
| Screenshot | Build in Public (n8n, resultados) |
| Carrossel | Authoritative (playbooks, listas) |
| Foto pessoal | Personal |
| Infográfico | Authoritative (dados, estatísticas) |

**Ferramentas:** Canva, Nano Banana, Pinterest (referências)

---

## Revisão Final

- [ ] Hook para o scroll em 2 linhas
- [ ] Body assertivo e escaneável
- [ ] CTA claro
- [ ] Visual definido
- [ ] Atrai meu ICP (não é vaidade)

---

> "Conteúdo que não atrai seu ICP é vaidade, não estratégia."
